# Front-end Level Up
### Daftacademy - Spring 2020
This project has been created as a part of course Daftacademy 
Front-end Level Up - Spring 2020.

## Technologies 
- Webpack
- NPM
- SCSS with BEM
- Bootstrap v4
- JavaScript (ES6+) 

## Run via Webpack Dev Server

```
$ npm run start
```

## Build from source

```
$ npm run build
```
